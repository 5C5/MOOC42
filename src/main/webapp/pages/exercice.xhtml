Consigne de l'exercice :

Le but de cet exercice est de determiner les valeurs d'entrée (en vert) qui permettent d'aboutir
à la sortie indiquée (en rouge) en fonction des portes logiques (en bleu ) présentes dans le circuit.

Pour modifier une entrée, cliquer sur le cercle la contenant.

Pour vérifier les réponses, cliquer sur vérifier.
